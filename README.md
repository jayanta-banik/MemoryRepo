# MemoryRepo

MemoryRepo is a CLI tool intended to help build memory for code repositories.

Initial usage:

```powershell
memoryrepo .
```

Output:

```text
Hello from MemoryRepo
```

The CLI accepts:

```text
memoryrepo [path/to/repo] [path/to/memory]
```

The memory path is optional and defaults to `./.RepoMemory`.
