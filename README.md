# MemorizeRepo

MemorizeRepo is a CLI tool intended to help build memory for code repositories.

Initial usage:

```powershell
memorizerepo .
```

Output:

```text
Hello from MemorizeRepo
```

The CLI accepts:

```text
memorizerepo [path/to/repo] [path/to/memory]
```

The memory path is optional and defaults to `./.RepoMemory`.
